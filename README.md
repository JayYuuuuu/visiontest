# 趣味视力测试

## 项目简介

这是一个专为幼儿设计的趣味视力测试网页应用，特别适合4岁左右的小朋友。通过有趣的动物主题游戏，可以对小朋友的视力进行初步筛查，帮助家长了解孩子的视力发育情况。

⚠️ 注意：本应用不能替代专业的医疗视力检查。如果发现孩子视力有异常，请及时咨询专业眼科医生。

## 使用方法

1. 打开`index.html`文件，在浏览器中运行测试
2. 将设备（推荐使用iPad或大屏幕手机）放在距离小朋友**2米远**的位置
3. 选择其中一种测试类型开始测试
4. 引导小朋友完成测试，根据他们能看清的最小图案大小评估视力
5. 测试结果页面可以打印保存

## 测试类型

本应用包含五种测试类型，每种测试都设有不同难度级别，图形大小会逐渐缩小：

1. **动物识别测试**：显示动物表情，让小朋友说出这是什么动物
2. **动物朝向测试**：动物看向某个方向，小朋友需要指出方向
3. **找出缺失部分**：显示缺少某个部位的动物，小朋友需要指出缺少了什么
4. **动物配对游戏**：让小朋友从多个选项中找出与样例相同的动物
5. **找不同游戏**：四个动物中有一个与众不同，小朋友需要找出来

## 测试结果解读

测试完成后会显示小朋友能够看清的最小图案尺寸，并给出相应的视力评估：

- **0.8 rem或更小**：视力非常好，相当于成人正常视力
- **0.8-1 rem**：视力很好，相当于成人较好的视力
- **1-2 rem**：视力良好，在正常范围内
- **2-3 rem**：视力正常，但建议1-2年后再次检查
- **3-4 rem**：视力略低于正常水平，建议半年后复查
- **大于4 rem**：建议咨询专业眼科医生

## 技术说明

- 纯前端实现，无需服务器
- 使用HTML5、CSS3和原生JavaScript开发
- 适配各种屏幕尺寸（响应式设计）
- 所有测试数据保存在本地，不会上传任何信息

## 推荐使用环境

- 推荐设备：iPad或大屏幕手机（如iPhone 15 Pro Max）
- 推荐浏览器：Chrome、Safari或其他现代浏览器
- 测试环境：光线充足的室内环境
- 测试距离：严格保持2米距离
